## 🚀 작업 워크플로우

### 프론트엔드 작업
```bash
cd D:\ahp
npm start                    # 개발 서버 실행
npm run build               # 프로덕션 빌드
git add . && git commit     # 변경사항 커밋
git push origin main        # GitHub Pages 배포
```

### 백엔드 작업
```bash

git push origin main        # Render 자동 배포
```
