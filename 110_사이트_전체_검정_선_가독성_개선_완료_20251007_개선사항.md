## 🔍 문제 분석

### 검정 테두리의 가독성 문제
```css
/* 문제가 있던 기존 코드 */
@media (prefers-contrast: high) {
  :root {
    --border-primary: #000000; /* 순수 검정 - 너무 강함 */
  }
}

@media print {
  .card {
    border: 1px solid black; /* 하드코딩된 검정 테두리 */
  }
}
```

### 가독성 저해 요소
1. **순수 검정색 (#000000)**: 과도한 대비로 시각적 피로감 유발
2. **하드코딩된 검정 테두리**: 일관성 없는 색상 시스템
3. **고대비 모드에서 극단적 대비**: 접근성 향상 목적이지만 오히려 가독성 저해
