- **즉시 행동 유도**: 명확한 "새 프로젝트 생성" 버튼

```
AHP for Paper 홈페이지

└── 서비스 소개 (토글)
    ├── 핵심 기능 소개
    ├── 활용 사례
    └── 시작하기 CTA
```

#### 5.2 개인 서비스 구조
```
개인 서비스 대시보드
├── 사용량 현황 (0/10 프로젝트, 0/50 사용자)
├── 결제 버튼
├── 통계 카드 (모두 0으로 초기화)
└── 프로젝트 섹션
    ├── 빈 상태: AHP 과정 안내 + 생성 버튼
    └── 프로젝트 목록: 생성된 프로젝트들
```
